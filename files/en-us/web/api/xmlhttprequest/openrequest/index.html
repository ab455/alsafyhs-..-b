{{APIRef("XMLHttpRequest")}}{{non-standard_header}}

This Mozilla-specific method is available only from within privileged code, and is only called from a C++ context in order to initialize an XMLHttpRequest. To initialize a request from JavaScript code, use the standard {{domxref("XMLHttpRequest.open", "open()")}} method instead.
